# Ashdata 注意力雷達

追蹤市場敘事的儀表板：新聞注意力、敘事異常偵測、新聞與價格的領先關係、頭條語料分類。

追蹤對象：**黃金 XAU** ・ **美股**（自動掃描候選池，取近 7 天語料聲量前三名）・ **比特幣 BTC**

## 快速開始

```powershell
pip install -r requirements.txt

# 1. 更新數據（首次約 5~10 分鐘；之後有快取會快很多）
python radar.py

# 2. 打開儀表板
start ui\index.html        # Windows
# open ui/index.html       # macOS
```

> 上游 API 有限流。腳本內建磁碟快取（`cache/`）與自動退避重試，
> 如果中途被中斷，直接重跑即可續傳，不會重複請求已成功的部分。

## 儀表板功能

- **三分頁**：黃金 / 美股 / 比特幣，分頁紅點 = 該資產有警報
- **美股聲量排行**：掃描候選池，近 7 天語料聲量前三名以橫條呈現，點擊展開完整數據
- **時間週期**：7D / 30D / 90D / ALL 一鍵切換
- **語料分類**：近 7 天頭條逐篇分類（看多/中性/看空），點擊開原文
- **日夜模式**：日間牛皮紙質感 / 夜間暖暗色，跟隨系統偏好，可手動切換
- **顯示設定**：字體大小四檔、文字對比三檔，設定記在瀏覽器
- **數據新鮮度**：右上角顯示最後更新時間，超過 24 小時提醒重跑

各指標的意義在儀表板的 ⓘ 圖示上有說明。

## 修改追蹤標的

編輯 [radar.py](radar.py) 頂部的 `XAU` / `BTC` / `US_CANDIDATES` 設定；
`US_TOP_N` 控制美股取前幾名。

## 部署到 GitHub Pages

`ui/` 是純靜態頁面。把倉庫推上 GitHub 後，到 Settings → Pages
選擇分支與 `/ui` 目錄（或用 Actions 把 `ui/` 發佈出去）即可。
頁面顯示的數據是你 commit 當下的 `ui/data.js`；要更新就重跑
`python radar.py` 後再 commit + push。

## 注意事項

- 警戒線會隨歷史數據自動校準，市場風格切換期誤報率會上升
